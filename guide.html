📖 How to Use

Chat with Velchat

No account. No download. Just open the site, share a one-time code with someone, and start chatting — encrypted directly between your browsers.

How it works
Velchat connects your browser directly to another person's browser — no server in the middle, no message storage, ever. Both of you visit this site. One person is the Host, the other the Peer. You swap a short one-time code to link up (via WhatsApp, Telegram, iMessage — anything). Once connected, the site is no longer involved. You're talking directly.
💡 Think of it like this
The code exchange is a one-time handshake. After that, your messages travel directly between browsers — like a private phone call. The website just helped you dial.
Connecting — step by step

Agree with the other person on who is Host and who is Peer — it doesn't matter which. Then follow the steps for your role. Keep a messaging app (WhatsApp, Telegram, etc.) open on the side to pass the codes between you.

🖥️ You are the Host
1
Click "I'M HOST" at the top of the page
2
A code appears in the box. Click "COPY"
3
Send that code to the Peer over any messaging app
4
Wait for the Peer to send you back their response code
5
Paste their code in the "Paste peer's answer" box and click "COMPLETE CONNECTION"
📱 You are the Peer
1
Click "I'M JOINING" at the top of the page
2
Paste the Host's code into the box that appears
3
Click "CONNECT" — your response code will appear
4
Click "COPY" and send that code back to the Host
5
You're done — once the Host completes their side, chat opens automatically ✓
✓ How you know it worked
The small circle in the top-left turns solid green and glows . The message input at the bottom becomes active. You'll see a system message in the chat confirming the encrypted connection is live.
Sending messages & files
✍
Messages

Click 📎 FILE and choose a file from your device. The other person receives a download link directly in chat. The file goes straight to their browser — never through a server.

Common questions
Why do I have to exchange a code? Can't it connect automatically?
Because there's no server doing the introduction for you. Most chat apps use a central server to connect two users behind the scenes. Velchat skips that entirely — so you pass the introduction data yourself. It's a one-time thing per session and takes about 30 seconds.
Are my messages stored anywhere on the site?
No. Messages are never sent to or stored on any server. They live only in your browser's memory while the tab is open. Close or refresh the tab and everything is permanently gone.
Is it safe to send the connection code over WhatsApp?
Yes. The code only contains the technical details needed to establish the connection — like your browser's network address. It contains no messages and no encryption keys. Someone who intercepts only the code cannot read your chat.
What happens if the connection drops mid-chat?
The green dot turns red and you'll see a warning. You'll need to repeat the setup — click Host or Join again and exchange new codes. Your previous messages stay visible in your tab until you refresh.
Does this work on mobile?
Yes. Velchat works in any modern mobile browser — Chrome, Safari, Firefox. The easiest flow on mobile is: copy the code, switch to WhatsApp or Telegram, paste and send it to the other person, then switch back to Velchat.
Can more than two people join?
No. Velchat is a private 1-to-1 channel — one Host, one Peer. Group chat is not supported.
What happens when I close the tab?
The connection ends immediately and all messages are erased from both sides. There's no history, no account, nothing saved. If you want to keep a record, use the Export button before closing.
⚠️ Before you close
If you need to keep a record of the conversation, hit EXPORT before closing the tab. Once the tab is closed, the chat is gone forever — by design.
